# Mevsimsel-Salg-n-Tahmini
Halka açık verilerle mevsimsel salgınlar için erken uyarı sistemi prototipi. Python (Pandas, Scikit-learn) ile Google Trends &amp; hava durumu verilerini analiz ederek yüksek riskli haftaları tahmin eden bir Lojistik Regresyon modeli eğitir. EDA'dan modellemeye tam bir veri bilimi iş akışı içerir.
